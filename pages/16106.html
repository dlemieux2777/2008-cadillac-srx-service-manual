Rear Number 1 Seat Back Release Latch Replacement



Rear Number 1 Seat Back Release Latch Replacement

Removal Procedure





1. Remove the screw (1) from the release assembly.
2. Disconnect the cable from the release assembly.
3. Remove the release assembly.



Installation Procedure

1. Connect the release cable to the release latch assembly.





Notice: Refer to Fastener Notice (Fastener Notice) .

2. Install the latch assembly to the seat cushion frame with the mounting screw (1).

Tighten the latch assembly screw to 2 N.m (18 lb in).
