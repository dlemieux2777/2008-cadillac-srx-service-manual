Transfer Case Assemble



Transfer Case Assemble


Tools Required
* J 5590 Pinion Bearing Race Installer - Rear
* J 8092 Driver Handle
* J 24433 Pinion Cone and Side Bearing Installer
* J 36850 Transjel Lubricant
* J 46268 Output Seal Installer
* J 46269 Input Seal Installer







Important:
* Lubricate all bearings and bearing journals with transfer case fluid during installation.

* Use a hammer and a brass drift only on the outer bearing race.

* Ensure the bearing is kept square to the bore while installing.



1. Install the input shaft bearing (117) into the case (1).





Important:
* Use a hammer and a brass drift only on the outer bearing race.

* Ensure the bearing is kept square to the bore while installing.



2. Install the front output shaft bearing (113) into the case (1).




3. Using the J 46269 (2) and the J 8092 (1), install the input shaft oil seal (13).




4. Using the J 46268 (2) and the J 8092 (1), install the front output shaft oil seal (14).





Notice: Refer to Fastener Notice (Fastener Notice) .


Important: While tightening the snubber bolts, position the snubber to the outside of the case, so when assembled, the snubber is in the farthest position away from the chain.


5. Install the chain snubber (3).
6. Apply sealant GM P/N 12345382 (Canadian P/N 10953489), or equivalent, to the threads of the bolts (4).

Tighten the bolts to 7 N.m (62 lb in).





7. Install the vent baffle (10).
8. Apply sealant GM P/N 12345382 (Canadian P/N 10953489), or equivalent, to the threads of the bolt (9).

Tighten the bolt to 7 N.m (62 lb in).






Important: The planetary differential gears are timed to the sun gears. If the gears are not timed properly, the differential will not rotate without binding - 40/60 ratio.


9. With the thrust washer side of the planetary differential facing down, align the marks on the pinion gears. Position the alignment marks with an area on the planetary differential that can be reference for all gears.





Important: The input shaft (118) must be installed to the carrier (111) with the raised bosses (1) facing up and the thrust washer groove facing down. Reversing this causes a mis-build, and the wrong torque split of 60/40.


10. Use a press to install the input shaft assembly to the carrier (111).




11. Lubricate the thrust washer (100) with J 36850 or equivalent.
12. Install the thrust washer over the rear of the input shaft, into the groove of the carrier (111).
13. Insert the sun gear (101) into the carrier.




14. If the pinion gears are properly aligned, the sun gear (101) carrier assembly will rotate freely.
15. Remove the sun gear (101) from the carrier (111).




16. Assemble the front output shaft (112) to the case (1).
17. Assemble the snap ring (114).




18. Assemble the chain (109), upper gear of the planetary assembly, and lower sprocket (115) to the case (1).




19. Use the J 5590 and a press to fully seat the lower sprocket (115) to the front output shaft, while guiding the input shaft of the planetary assembly, until it is seated to the case.





Important:
* Use a hammer and a brass drift only on the outer bearing race.

* Ensure the bearing is kept square to the bore while installing.



20. Install the rear output shaft bearing (104).




21. Using the J 46268 (2) and the J 8092 (1), install the rear output oil seal (7).





Important:
* Use a hammer and a brass drift only on the outer bearing race.

* Ensure the bearing is kept square to the bore while installing.



22. Install the inner output shaft bearing (103).





Important:
* Use a hammer and a brass drift only on the outer bearing race.

* Ensure the bearing is kept square to the bore while installing.



23. Install the front output shaft inner bearing (116).




24. Install the rear output shaft (106) by pushing it in through the bearings.
25. Install the rear output shaft snap ring (102) in the lower groove of the output shaft (106).




26. Use the J 24433 and a press to install the sun gear (101) to the output shaft.





Important:
* Ensure both the sealing surfaces on the case and cover are free of dirt, oil, and cleaning solvent.

* Ensure the dowel pins (5) are installed in the cover (2).



27. Install the dowel pins (5) into the cover (2), if necessary.
28. Apply a 3.175 mm (1/8 in) bead of RTV sealant GM P/N 12345739 (Canadian P/N 10953541), or equivalent, to the mating surfaces of the cover (2).




29. Insert the dowel pin (3), if removed.
30. Lower the cover (2) over the case (1). Turn the rear output shaft in order to align the splines and the shaft into the bearing in the cover.





Important: The bolts are self-tapping. They must be hand started. Do NOT use power-assisted tools to install the bolts (11).


31. Install the case bolts (11).

Tighten the case bolts to 25 N.m (18 lb ft) in a criss-cross pattern.





32. Install the vent breather by pushing it into the hole. Align the flat on vent with the flat on the case.
33. Lightly pull on the vent to ensure proper installation.




34. Apply pipe sealant GM P/N 12346004 (Canadian P/N 10953480), or equivalent, to the threads of the drain plug (8).
35. Install the drain plug (8).

Tighten the drain plug to 25 N.m (18 lb ft).





36. Fill the transfer case with manual transmission fluid GM P/N 88861800 (Canadian P/N 88861801). Fill to the bottom of fill plug threads. Ensure the vehicle is on a level surface. Refer to Approximate Fluid Capacities (Capacity Specifications) .
37. Apply pipe sealant GM P/N 12346004 (Canadian P/N 10953480), or equivalent, to the threads of the fill plug.
38. Install the fill plug (6).

Tighten the fill plug to 25 N.m (18 lb ft).





39. Install the dampener (300) and the bolts (16).

Tighten the bolts to 27 N.m (20 lb ft).
