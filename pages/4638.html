Utility Storage Compartment: Service and Repair



Rear Storage Compartment Replacement

Removal Procedure

1. Open the storage compartment.




2. Remove the removable storage bin (1) from the cargo assembly.
3. Remove the storage compartment bolts (2).
4. Remove the storage compartment assembly.



Installation Procedure





1. Insert the storage compartment front lower hooks (3) into the floor.

Notice: Refer to Fastener Notice (Fastener Notice) .

2. Install the retaining bolts (2).

Tighten the screw to 45 N.m (33 lb ft).

3. Install the removable storage bin (1).
