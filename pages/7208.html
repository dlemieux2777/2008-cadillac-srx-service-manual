B1023



DTC B1023

Diagnostic Instructions

* Perform the Diagnostic System Check - Vehicle (Initial Inspection and Diagnostic Overview) prior to using this diagnostic procedure.
* Review Strategy Based Diagnosis (Initial Inspection and Diagnostic Overview) for an overview of the diagnostic approach.
* Diagnostic Procedure Instructions (Initial Inspection and Diagnostic Overview) provides an overview of each diagnostic category.



DTC Descriptor
DTC B1023 00
- Integral Switch Performance




Circuit/System Description

The driver door module (DDM) and front passenger door module (PDM) each contain switches that control power door lock and window functions. The DDM also contains the control switch for the power outside rear view mirrors. These switches are integrated within the door control modules and are not serviceable as separate components. The DDM and PDM monitor these switch inputs for internal faults and set DTC B1023 00 if any are detected.


Conditions for Running the DTC

Battery voltage must be between 9-16 volts.


Conditions for Setting the DTC

* DTC B1023 00 will set when the DDM or PDM detects multiple inputs from a window or door lock switch.
* Condition must be present for greater than 30 seconds.



Action Taken When the DTC Sets

The DDM or PDM ignores inputs from the malfunctioning switch.


Conditions for Clearing the DTC

* The DTC will be current for as long as the fault is present.
* When the fault is no longer present, the DTC will be a history status code.

Circuit/System Verification

1. Ignition ON, install a scan tool and display the inputs data list of the affected door module. All integrated switch inputs should display INACTIVE while not in use.

• If any of the integrated switch inputs are displayed as ACTIVE while not in use, replace the door control module associated with the malfunctioning switch.

2. Clear all DTCs and operate the integrated switches at each door control module within the Conditions for Running the DTC.

• If the DTC resets, replace the appropriate door control module.
