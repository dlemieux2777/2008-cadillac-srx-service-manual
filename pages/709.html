128. Engine Flywheel Installation



Engine Flywheel Installation


Tools Required
* EN 46326 Flywheel Holding Tool
* J 45059 Angle Meter


1. Apply sealant, GM P/N 12346004 (Canadian P/N 10953480) or equivalent, to the engine flywheel bolts.




2. Place the engine flywheel in position on the crankshaft.
3. Slip one bolt into the engine flywheel.
4. Install the bolt in a location at the top of its rotation allowing the engine flywheel to hang in position.




5. Install the EN 46326 to the engine block.

Notice: Refer to Fastener Notice (Fastener Notice) .

6. Install the remaining engine flywheel bolts.

1. First Pass

Tighten the engine flywheel bolts to 30 N.m (22 lb ft).

2. Final Pass

Tighten the engine flywheel bolts an additional 50 degrees using the J 45059 .
