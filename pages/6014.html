Power Folding Seat Calibration



Power Folding Seat Calibration

Calibration Criteria

Rear seat module (RSM) calibration must be performed after the rear power folding seat system or RSM have been serviced or replaced. If the calibration procedure is not performed, the rear power folding seat will not operate.

1. Open the liftgate or the right rear passenger door.
2. Install a scan tool.
3. Clear all RSM DTCs before proceeding.
4. Navigate to the Rear Seat Module menu.
5. Select Special Functions menu item.
6. Select the Auto Calibration procedure and follow the directions displayed on the screen.
7. Verify the RSM has been calibrated by making the following scan tool selections:

* Rear Seat Module
* Data Display
* Data

8. The scan tool Calibrated parameter should display YES.
