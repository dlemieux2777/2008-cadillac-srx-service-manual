Prop Flange Assemble



Prop Flange Assemble

Tools Required

J 45012 Holding Fixture





1. Install the NEW output shaft assembly retainer (30).
2. Install the NEW prop shaft flange o-ring seal (32) into the prop shaft flange (31).
3. Install the prop shaft flange (31).
4. Put the transmission in Park.

Notice: Refer to Fastener Notice .

5. Install the NEW prop shaft flange thrust washer (33) and NEW prop shaft flange nut (34) using GM P/N 89021297.
6. Using J 45012 , tighten the prop flange nut.

Tighten the prop shaft flange nut (34) to 57 N.m (42 lb ft).
