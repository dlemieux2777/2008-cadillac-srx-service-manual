6L50/6L80/6L90 - Automatic Transmission



DTC P0711

Diagnostic Instructions

* Perform the Diagnostic System Check - Vehicle (Initial Inspection and Diagnostic Overview) prior to using this diagnostic procedure.
* Review Strategy Based Diagnosis (Initial Inspection and Diagnostic Overview) for an overview of the diagnostic approach.
* Diagnostic Procedure Instructions (Initial Inspection and Diagnostic Overview) provides an overview of each diagnostic category.



DTC Descriptors
DTC P0711
- Transmission Fluid Temperature (TFT) Sensor Performance




Circuit/System Description

The transmission fluid temperature (TFT) sensor is located inside of the transmission control module (TCM) assembly and has no serviceable parts. The TFT sensor provides transmission fluid temperature to the TCM. This fault is handled inside the TCM and no external circuits are involved.


Conditions for Running the DTC


Condition 1
* No DTCs P0711, P0716, P0717, P0722, or P0723.
* DTC P0711 not passed this key ON.
* Ignition voltage is between 8.6-18.0 volts.
* The engine coolant temperature (ECT) is valid.
* The TCM internal temperature is -49°C to +169°C (-56°F and +336°F).
* The engine speed is 500-7500 RPM for at least 5 seconds.
* The vehicle speed is greater than 8 km/h (5 mph) for 300 seconds cumulative.
* TCC slip speed is greater than 120 RPM for 300 seconds cumulative.
* The TFT is between -40°C and +21°C (-40°F and +70°F).
* The TFT has changed by less than 2.0°C (3.6°F) in 100 seconds.
* The ECT is greater than 70°C (158°F).
* The ECT has increased by 55°C (99°F) since startup.
* The TCM TFT sensor is between -50°C and +169°C (-58°F and +336°F) at start up.



Condition 2
* No DTCs P0711, P0716, P0717, P0722, or P0723.
* DTC P0711 not passed this key ON.
* Ignition voltage is between 8.6-18.0 volts.
* The ECT is valid.
* The TCM internal temperature is -49°C to +169°C (-56°F and +336°F).
* The engine speed is 500-7500 RPM for at least 5 seconds.
* The vehicle speed is greater than 8 km/h (5 mph) for 300 seconds cumulative.
* The ECT is greater than 70°C (158°F).
* The ECT has increased by 55°C (99°F) since startup.
* The TCM TFT sensor is between 129-170°C (264-338°F) at start up.



Condition 3
* Ignition voltage is between 8.6-18.0 volts.
* The engine speed is 500-7500 RPM for at least 5 seconds.



Condition 4
* No DTCs P0711, P0716, P0717, P0722, or P0723.
* DTC P0711 not passed this key ON.
* Ignition voltage is between 8.6-18.0 volts.
* The ECT is valid.
* The ECT is between -39°C and +149°C (-36°F and +300°F).
* The TCM internal temperature is -49°C to +169°C (-56°F and +336°F).
* Accelerator pedal position signal is valid.
* Accelerator pedal position is between 8-100 percent.
* The engine speed is 500-7500 RPM for at least 5 seconds.
* The TCM must receive a valid torque signal from the ECM.
* Engine torque is between 50-1492 N.m (36-1100 lb ft).
* Vehicle speed is between 8-511 km/h (5-317 mph).
* No transmission shift default actions active.



Conditions for Setting the DTC


Condition 1
* Vehicle speed is greater than 8 km/h (5 mph) for 300 seconds cumulative.
* TCC slip speed is greater than 120 RPM for 300 seconds cumulative.
* The TFT is between -50°C and +21°C (-58°F and +70°F).
* The TFT has changed by less than 2.0°C (3.6°F) in 100 seconds.
* The ECT is greater than 70°C (158°F).
* The ECT has increased by 55°C (99°F) since startup.



Condition 2
* Vehicle speed is greater than 8 km/h (5 mph) for 300 seconds cumulative.
* TCC slip speed is greater than 120 RPM for 300 seconds cumulative.
* The TFT is between 129-170°C (264-507°F).
* The TFT has changed by less than 2.0°C (3.6°F) in 100 seconds.
* The ECT is greater than 70°C (158°F).
* The ECT has increased by 55°C (99°F) since startup.


Condition 3

The TFT power up temperature changes by greater than or equal to 20°C (36°F) 14 times over 7 seconds.

Condition 4

The TFT has not changed more than 20°C (36°F) during the following conditions:

* The TFT temperature at start up is -50°C (-58°F) for 2500 seconds (41.6 minutes).
* The TFT temperature at start up is -25°C (-13°F) for 1000 seconds (16.6 minutes).
* The TFT temperature at start up is -10°C (14°F) for 800 seconds (13.3 minutes).
* The TFT temperature at start up is -5°C (23°F) for 520 seconds (8.6 minutes).
* The TFT temperature at start up is 20°C (68°F) for 200 seconds (3.3 minutes).



Action Taken When the DTC Sets

* DTC P0711 is a Type C DTC.
* The TCM freezes transmission adaptive functions.
* The TCM allows the vehicle to operate in transmission protection mode.



Conditions for Clearing the DTC

DTC P0711 is a Type C DTC.

Circuit/System Verification

1. Start the engine and operate the vehicle at 64 km/h (40 mph) while observing the TCM start up temperature sensor and the TFT sensor.
2. The system is operating normally if the TCM start up temperature is within 10°C (18°F) of the TFT temperature.



Circuit/System Testing

Clear the DTC and operate the vehicle under the conditions for running and setting the DTC. Verify that the DTC does not reset.

If the DTC resets, replace the control solenoid (w/body and TCM) valve assembly.



Component Testing

Perform the Control Solenoid Valve and Transmission Control Module Assembly Inspection (Control Solenoid Valve and Transmission Control Module Assembly Inspection) .


Repair Instructions

Perform the Diagnostic Repair Verification (Verification Tests) after completing the diagnostic procedure.

* Control solenoid (w/body and TCM) valve assembly replacement. Refer to Control Module References (Programming and Relearning) for replacement, setup, and programming.
* Perform the Service Fast Learn Adapts (Programming and Relearning) if internal transmission repairs are performed.
