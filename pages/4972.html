Front Seat Recliner Replacement



Front Seat Recliner Replacement

Removal Procedure

1. Remove the seat. Refer to Bucket Seat Replacement (Bucket Seat Replacement) .
2. Remove the seat back frame. Refer to Front Seat Back Frame Replacement (Front Seat Back Frame Replacement) .
3. Remove the seat cushion. Refer to Front Seat Cushion Replacement (Front Seat Cushion Replacement) .




4. Disconnect the recliner motor electrical connectors.
5. Remove the seat axle retainers (3).
6. Pull the seat track sides apart in order to separate the seat track sides.
7. Remove the nuts (1) securing the recliner to the seat track.



Installation Procedure






Notice: Refer to Fastener Notice (Fastener Notice) .

1. Install the nuts (1) securing the recliner to the seat track.

Tighten the nut to 65 N.m (18 lb ft).

2. Locate the axles and the seat position drive cable (2) in order to connect the seat track sides.
Important: Insure that the seat position motor cable is fully seated into the drive gear.



3. Install the seat axle retainers (3).
4. Connect the recliner motor electrical connectors.
5. Install the seat cushion. Refer to Front Seat Cushion Replacement (Front Seat Cushion Replacement) .
6. Install the seat back frame. Refer to Front Seat Back Frame Replacement (Front Seat Back Frame Replacement) .
7. Install the seat. Refer to Bucket Seat Replacement (Bucket Seat Replacement) .
8. Calibrate the seat. Refer to Memory Seat Calibration (Programming and Relearning) .
