Output Speed Sensor Installation



Output Speed Sensor Installation





1. Install the output speed sensor (42) into the case.

Notice: Refer to Fastener Notice .

2. Install the output speed sensor bolt (43) into the case.

Tighten the output speed sensor bolt (43) to 11 N.m ( 8 lb ft).
