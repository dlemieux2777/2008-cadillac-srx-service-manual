Rear Knuckle Alignment Link: Service and Repair



Adjustment Link Replacement

Removal Procedure

1. Raise and support the vehicle. Refer to Lifting and Jacking the Vehicle (Service and Repair) .
2. Remove the tire and wheel. Refer to Tire and Wheel Removal and Installation (Service and Repair) .




3. Remove the adjustment link to knuckle mounting bolt.
4. Remove the adjustment link to frame mounting nut.
5. Remove the adjustment link to frame mounting bolt.
6. Remove the adjustment link from the vehicle.



Installation Procedure





1. Install the adjustment link to the vehicle.

Notice: Refer to Fastener Notice (Fastener Notice) .


Important: Loosely install all fasteners before tightening.


2. Install the adjustment link to knuckle mounting bolt.

Tighten the bolt to 160 N.m (118 lb ft).

3. Install the adjustment link to frame mounting bolt.
4. Install the adjustment link to frame mounting nut.

Tighten the bolts and nut to 170 N.m (125 lb ft).

5. Install the tire and wheel. Refer to Tire and Wheel Removal and Installation (Service and Repair) .
6. Lower the vehicle.
7. Adjust the rear toe. Refer to Rear Toe Adjustment (Service and Repair) .
