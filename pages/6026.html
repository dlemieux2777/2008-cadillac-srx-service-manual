Window Regulator Motor Programming and Setup



Window Regulator Motor Programming and Setup

Programming Criteria

Window regulator motor programming must be performed after a window regulator motor has been replaced or is disconnected from battery voltage. When the reprogramming procedure has been completed, the PWR WNDWS fuse must be removed for at least 30 seconds.


When a window regulator motor is disconnected from battery voltage, it must be reprogrammed in order for the express-UP or express-DOWN feature to operate properly. Keep in mind, that if something is obstructing the window from opening or closing and you continue programming, the window will only express-UP or express-DOWN to the spot where the obstruction occurred. If an obstruction comes in contact with the window during the programming procedure, the procedure will have to be repeated.


Programming Procedure

1. Install a scan tool.
2. Navigate to the Body menu item.
3. Select the appropriate (Driver or Passenger) Door Switch menu item.
4. Select Special Functions.
5. Select Power Window Motor Programming and follow the directions displayed on the screen.
6. When the window motor programming procedure has been successfully completed, remove the PWR WNDWS fuse from the left rear fuse block for at least 30 seconds.
7. Install the PWR WNDWS fuse.
8. Press and hold the power window switch until the window is fully DOWN.
9. Continue holding the power window switch for approximately 3 seconds.
10. Press and hold the power window switch until the window is fully UP.
11. Continue holding the power window switch for approximately 3 seconds.
12. The power window regulator motor is now programmed for the express function.
