Antitheft Motion Sensor: Service and Repair



Theft Deterrent Sensor Replacement

Removal Procedure





1. Remove the overhead console. Refer to Overhead Console Courtesy Lamp Replacement (Overhead Console Courtesy Lamp Replacement) .




2. Disconnect the electrical connector.
3. Release the tabs securing the sensor to the overhead console.



Installation Procedure





1. Install the sensor securing the tabs to the overhead console.
2. Connect the electrical connector.




3. Install the overhead console. Refer to Overhead Console Courtesy Lamp Replacement (Overhead Console Courtesy Lamp Replacement) .
