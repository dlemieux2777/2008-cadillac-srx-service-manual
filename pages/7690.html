P1761



DTC P1761

Diagnostic Instructions

* Perform the Diagnostic System Check - Vehicle (Initial Inspection and Diagnostic Overview) prior to using this diagnostic procedure.
* Review Strategy Based Diagnosis (Initial Inspection and Diagnostic Overview) for an overview of the diagnostic approach.
* Diagnostic Procedure Instructions (Initial Inspection and Diagnostic Overview) provides an overview of each diagnostic category.



DTC Descriptor
DTC P1761
- Up and Down Shift Switch Signal Circuit




Circuit/System Description

The TAP Shift Switch Signal system allows the driver to manually shift gears by using the Upshift and Downshift positions located on the automatic transmission shift Lever. When TAP mode is selected, the shift switch input signals to the body control module (BCM). This code sets when the rolling count of the TCM does not match the same value sent from the BCM.


Conditions for Running the DTC

The engine run time is greater than 5 seconds.


Conditions for Setting the DTC

The TCM detects an error in the rolling count value that does not match with the BCM.


Action Taken When the DTC Sets

* DTC P1761 is a Type C DTC.
* The TCM disables TAP Shift operation.



Conditions for Clearing the DTC

DTC P1761 is a Type C DTC.
