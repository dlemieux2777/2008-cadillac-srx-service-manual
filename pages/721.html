140. Exhaust Manifold Installation - Right Side



Exhaust Manifold Installation - Right Side (AWD)






Notice: Refer to Fastener Notice (Fastener Notice) .

1. Install the right engine mount bracket.
2. Install the right engine mount bracket bolts.

Tighten the engine mount bracket bolts to 60 N.m (44 lb ft).





3. Install the right engine mount.
4. Install the right engine mount upper nut.

Tighten the engine mount upper nut to 80 N.m (59 lb ft).





5. Install the right exhaust manifold studs to the right cylinder head.

Tighten the exhaust manifold studs to 6 N.m (53 lb in).





6. Install the NEW right exhaust manifold gasket.




7. Install the right exhaust manifold.
8. Loosely install the right exhaust manifold bolts and nuts to the right cylinder head.

Right:




9. Tighten the exhaust manifold bolts and nuts in the sequence shown.

Tighten the exhaust manifold bolts and nuts to 25 N.m (18 lb ft).





10. Install the right exhaust manifold heat shield.
11. Install the right exhaust manifold heat shield bolts.

Tighten the exhaust manifold heat shield bolts to 10 N.m (89 lb in).
