13. Crankshaft Balancer Removal



Crankshaft Balancer Removal


Tools Required
* EN 46326 Flywheel Holding Tool
* J 24420-C Harmonic Balancer Puller
* J 38416-2 Crankshaft Button






1. Install the EN 46326 .




2. Remove the crankshaft balancer bolt.




3. Place the J 38416-2 into the end of the crankshaft.
4. Install the J 24420-C on the crankshaft balancer.




5. Remove the crankshaft balancer using the J 24420-C .
