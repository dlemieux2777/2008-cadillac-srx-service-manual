Backup Lamp: Service and Repair



Backup Lamp Replacement

Removal Procedure





1. Open the liftgate.
2. Remove the liftgate applique. Refer to Liftgate Applique Replacement () .
3. Remove the back up lamp socket (1) from the liftgate applique by turning the socket (1) counterclockwise.
4. Remove the back up lamp bulb from the socket.



Installation Procedure





1. Install the back up lamp bulb into the socket.
2. Install the back up lamp socket (1) into the liftgate applique by inserting it and turning it clockwise.
3. Install the liftgate applique to the liftgate. Refer to Liftgate Applique Replacement () .
4. Close the liftgate.
