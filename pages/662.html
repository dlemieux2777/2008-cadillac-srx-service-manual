83. Engine Front Cover Assemble



Engine Front Cover Assemble






Notice: Refer to Fastener Notice (Fastener Notice) .

1. Install the engine front cover fitting.

Tighten the engine front cover fitting to 25 N.m (18 lb ft).





2. Install the NEW thermostat gasket onto the thermostat.




3. Install the thermostat and gasket into the thermostat housing.




4. Install the thermostat housing.
5. Install the thermostat housing bolts.

Tighten the thermostat housing bolts to 10 N.m (89 lb in).





6. Install the NEW water pump O-ring.




7. Install the water pump.
8. Install the water pump bolts.

Tighten the water pump bolts to 10 N.m (89 lb in).
