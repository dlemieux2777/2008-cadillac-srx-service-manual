89. Intake Manifold Assemble



Intake Manifold Assemble






Notice: Refer to Fastener Notice (Fastener Notice) .

1. Install the vacuum boost fitting.

Tighten the vacuum boost fitting to 2 N.m (18 lb in).





2. Install the manifold vacuum source cap.




3. Install the sight shield retainer ball studs.

Tighten the sight shield retainer ball studs to 7 N.m (62 lb in).






Important: Do not reuse the intake manifold-to-cylinder head sealing gaskets.


4. Install NEW intake manifold-to-cylinder head gaskets.
